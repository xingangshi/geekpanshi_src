2. 敏捷联盟

雪鸟会议的结果是创造了一个新术语，将所有这些方法置于同一屋檐下：敏捷。他们发布了一份宣言，概述了十二项核心原则，并成立了敏捷联盟，这是一家非营利性倡导组织，旨在向软件工程界宣传他们的新方法。

在过去十年中，敏捷运动取得了巨大成功。它的创始人已经成为有影响力的“思想领袖”，他们在世界各地的编程会议上发表演讲和主题演讲。
